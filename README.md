## 项目介绍

### install dependencies

```javascript
yarn install
```

### Compiles and hot-reloads for development

```javascript
yarn serve
```

### Compiles and minifies for production

```javascript
yarn build
```

### Lints and fixes files

```javascript
yarn lint
```
